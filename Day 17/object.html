Object fit Properties

cover property used the given width and height and fit that image in it, to cover that part remaining part of that image will be cut.
contain property used the given width and height , show the full image , without cutting the image and remaining width and height get remains blank.
fill property used the full given width and height and show full image, The image is resized to fill the given dimension. If necessary, the image will be stretched or squished to fit
